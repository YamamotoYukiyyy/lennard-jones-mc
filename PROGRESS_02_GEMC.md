# 進捗ログ 02: 3D GEMC 追加

- 日付: 2026-03-07
- 参照: `lennard_jones_mc/WORK_LOG_GEMC_3D_2026-03-07.md`

## 概要

3 次元 Lennard-Jones 流体に対する Gibbs Ensemble Monte Carlo を追加した。

## 実装方針

- 2 箱を独立した `BoxState` として管理
- 1 サイクル: 粒子変位 → 体積交換 → 粒子交換
- 共通のエネルギー計算や初期配置は `lj_core.py` を再利用

## 追加したファイル

- `lennard_jones_gemc_3d.py`

## 実装内容

### 状態管理（BoxState）

- 粒子座標 `pos`
- 箱の一辺 `box_size`
- 現在の全エネルギー `energy`
- 派生量: `volume`, `n_particles`, `density`

### move の実装

| move | 内容 |
|------|------|
| 粒子変位 | 箱内で粒子 1 個をランダム変位、Metropolis 判定 |
| 体積交換 | 全体積一定で V1↔V2 を変化、座標を一様スケール、GEMC 標準受理式 |
| 粒子交換 | 一方から削除、他方へランダム位置に挿入、挿入・削除エネルギーで受理判定 |

### 記録する履歴

- N1, N2 / rho1, rho2 / V1, V2 / E1, E2

### 可視化

- Box 1/2 最終配置、密度履歴、粒子数履歴を 1 枚の図にまとめて保存

## 主な判断

- まずは可読性重視で O(N²) 実装
- 体積交換は dV 提案で実装
- 近接リストやカットオフ最適化は当初は未導入
