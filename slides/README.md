# Marp スライド

このフォルダでは [Marp](https://marp.app/) を使ってスライドを Markdown で作成・編集できます。

## 手順の流れ（新規スライドを作る場合）

1. **テンプレートをコピー**: `_template.md` をコピーし、新しいファイル名（例: `my_slides.md`）で保存する
2. **フロントマターを編集**: ファイル先頭の `---` で囲まれた YAML で `title` / `description` を設定
3. **スライドを編集**: `---`（ハイフン3つ）でスライドを区切り、見出し・リスト・表・コード・数式を記述
4. **プレビュー**: コマンドパレット → 「Marp: Open Preview」でスライド表示を確認
5. **エクスポート**: プレビュー右上の「Export Slide Deck」から PDF / HTML / PPTX を出力

既存の `vibe_coding_slides.md` を編集するだけの場合は、上記の 3〜5 から行えばよいです。

## テンプレート（_template.md）

`_template.md` は発表向けに整理した Marp 用のひな形です。`vibe_coding_slides.md` の構成をベースにしています。

**含まれるパターン**

- **表紙** タイトル＋副題＋アジェンダ（箇条書き）
- **本題** モデル・背景（数式・箇条書き）
- **実行方法** 仮想環境の Python で実行＋スクリプト名
- **重要パラメータ** 2×2 のカード風の表（4点に絞る）
- **変更ログ** 「やったこと→内容→成果」「問題→対応→結果」の型、または **ラベル** 短文の並び
- **現在の仕様はこの3点** 番号リスト＋下部に `<small>` で参照先
- **Appendix** パラメータ一覧の表（詳細はここへ）

ファイル内の HTML コメントに、使い方とデザインのルール（1枚1メッセージ、表のカード化、参照先の置き方、推奨順序）も記載しています。新しいスライドを作るときはこのファイルをコピーして使うと、流れと見せ方を統一できます。

## 必要なもの

- **Marp for VS Code** 拡張機能（Cursor では VS Code 拡張が利用可能です）
  - 拡張機能パネルで「Marp」で検索 → **Marp for VS Code**（marp-team.marp-vscode）をインストール

## 使い方

1. `vibe_coding_slides.md` を Cursor で開く
2. スライドの区切りは `---`（ハイフン3つ）で行う
3. 数式は `$...$`（インライン）または `$$...$$`（ブロック）で記述

### Cursor でスライドプレビューを表示する

- **コマンドパレット**（`Cmd+Shift+P` / `Ctrl+Shift+P`）を開く
- 「**Markdown: Open Preview to the Side**」を実行 → 標準の Markdown プレビューが開く
- Marp 拡張が有効なら、プレビュー枠の**右上に 2 つのプレビューボタン**が出ます。**右側のアイコン**（スライド用）をクリックすると **Marp スライドプレビュー**に切り替わります
- または「**Marp: Open Preview**」で検索して実行（Marp 拡張が入っていれば表示されます）

**プレビューが出ない場合**

- ファイル先頭の `marp: true` が `false` になっていないか確認する
- 他の Markdown プレビュー拡張（Markdown Preview Enhanced など）が競合していることがあります。一時無効化して試す
- **HTML でエクスポート**: コマンドパレットで「**Marp: Export Slide Deck**」→ HTML を選ぶと、ブラウザでスライドを開いて確認できます

## エクスポート

- **PDF**: プレビュー右上の「Export Slide Deck」などから PDF を出力
- **HTML**: 同様に HTML としてエクスポート可能

## 設定（ファイル先頭の YAML）

| 項目 | 説明 | 例 |
|------|------|-----|
| `marp` | Marp を有効化 | `true` |
| `theme` | テーマ | `default`, `gaia`, `uncover` など |
| `size` | スライド比率 | `16:9`, `4:3` |
| `paginate` | ページ番号表示 | `true` |

詳細: [Marp - Directives](https://marp.app/docs/guide/directives)
